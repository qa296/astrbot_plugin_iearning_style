# astrbot_plugin_iearning_style（入乡随俗）

使bot能够从聊天中学习并模仿他人的说话方式。插件通过分析聊天记录，自动总结语言风格和语法特点，从而让机器人更具个性化和适应性。

## ✨ 功能特性

插件从聊天记录中提取**三层表征**，让机器人融入当前会话：

| 层级 | 本质 | 例子 | 
|------|------|------| 
| **通用表征** | 稳定的说话基调 | 语气活泼、喜欢玩梗、爱用短句 | 
| **情境表征** | 场景→行为模式 | 别人难过时→发猫猫图安慰；被夸时→谦虚回应 | 
| **梗释义**   | 具体梗/说法 + 释义 | awsl（啊我死了，用于表达被可爱到或惊喜到） | 

### 核心机制

- **自动学习**：静默监听所有聊天消息，无需手动触发
- **LLM 分析**：每轮进行一次 LLM 调用，输出三层表征
- **通用全量重写**：每次分析时 LLM 重新评估通用表征，保留合适的、加入新的、淘汰过时的
- **特定正则触发**：特定表征带 `trigger_regex`，只有用户消息匹配时才注入，不膨胀 prompt
- **情境 FIFO 管理**：情境表征固定容量，超限按最早写入淘汰；缓冲位（20%）在维护时尝试合并到通用/特定
- **熟练度延续**：连续多轮被确认的通用表征 proficiency 递增；特定表征 trigger_count 递增，达到阈值时提示 LLM 考虑升格为通用


## 📝 命令

- `风格状态` — 查看当前会话三层表征的统计概览
- `学习总结` — 手动触发当前会话的学习分析，无需等待定时任务
- `清空风格` — 清空当前会话所有学习到的表征
